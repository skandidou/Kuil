## BUDGET RECOMMENDATIONS

### Phase 1 - Test (Semaines 1-2)
- **Budget** : $50-100/jour
- **Focus** : Tester tous les ad sets, identifier les meilleurs performers
- **Métriques** : CTR, CPI (Cost per Install), Hook rate

### Phase 2 - Optimize (Semaines 3-4)
- **Budget** : $100-200/jour
- **Focus** : Scale les gagnants, tuer les perdants
- **Métriques** : CPI, Retention D1/D7, CAC

### Phase 3 - Scale (Mois 2+)
- **Budget** : $200-500/jour
- **Focus** : Lookalike audiences, retargeting
- **Métriques** : LTV:CAC ratio, Organic lift

## CREATIVE GUIDELINES

### Visuels
- **Couleurs** : Utiliser les couleurs de la marque Kuil
- **Style** : Clean, moderne, professional mais pas corporate
- **Screenshots** : Montrer l'app en action (Voice Signature, Calibration)

### Copy
- **Ton** : Direct, authentique, conversationnel
- **Longueur** : Court pour les hooks, détaillé pour le body
- **CTA** : Toujours mentionner "iOS" et "gratuit"

### Video
- **Hook** : Les 3 premières secondes sont cruciales
- **Format** : Privilégier vertical (9:16) pour mobile
- **Sous-titres** : Toujours inclure (80% regardent sans son)
